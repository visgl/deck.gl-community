# basemaps

[![NPM Version](https://img.shields.io/npm/v/@deck.gl-community/basemaps.svg)](https://www.npmjs.com/package/@deck.gl-community/basemaps)
[![NPM Downloads](https://img.shields.io/npm/dw/@deck.gl-community/basemaps.svg)](https://www.npmjs.com/package/@deck.gl-community/basemaps)

> _This is a work in progress_.

An implementation of a subset of the [Mapbox Style
Specification][mapbox_style_spec] with the intention of visualizing it using deck.gl.

The runtime API is exported from `@deck.gl-community/basemaps`, and the pure
style helpers are exported from `@deck.gl-community/basemaps/style-spec`.

For more information, see the online documentation.
