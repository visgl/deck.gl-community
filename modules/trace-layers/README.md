# @deck.gl-community/trace-layers

Private scaffold for the planned Tracevis open source landing in deck.gl-community.

This package currently reserves the final root, `trace-graph`, `deck-trace-layers`,
`trace-loaders`, and `tracevis` entrypoints. The reusable Tracevis source and real Tracevis example
application will land in follow-up work.

See the [Trace Layers docs](../../docs/modules/trace-layers/README.md) and the
[Tracevis placeholder example](../../examples/trace-layers/tracevis).
